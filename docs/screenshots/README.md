This directory contains screenshots for the QuMail README documentation.

2. **dashboard.png** - Main email inbox/dashboard view

## Recommended Specifications

- **Resolution**: 1920x1080 or higher for desktop screenshots
- **Format**: PNG (preferred) or JPG
- **File size**: Optimize images to under 500KB each
- **Content**: Ensure no sensitive/personal information is visible

## How to Take Screenshots

### Windows
- `Win + Shift + S` - Snipping Tool
- `Print Screen` - Full screen capture

### macOS
- `Cmd + Shift + 4` - Selection capture
- `Cmd + Shift + 3` - Full screen capture

### Browser DevTools
- F12 > Device toolbar for responsive screenshots
- Use "Capture screenshot" feature

## Tips for Good Screenshots

1. Use sample/demo data (not real emails)
2. Show the most important features
3. Ensure consistent window sizes
4. Highlight key UI elements if needed
5. Use dark mode for consistency with the app theme
